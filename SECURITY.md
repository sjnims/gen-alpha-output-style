# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities through GitHub's private vulnerability reporting feature:

1. Go to the [Security tab](https://github.com/sjnims/gen-alpha-output-style/security) of this repository
2. Click "Report a vulnerability"
3. Fill out the form with details about the vulnerability

You will receive a response acknowledging your report within 48 hours. We will work with you to understand the issue and develop a fix.

Please do not report security vulnerabilities through public GitHub issues.

## Scope

This plugin primarily transforms text output and does not handle sensitive data. However, we take all security reports seriously, including:

- Shell injection vulnerabilities in hook scripts
- Path traversal issues
- Any unexpected code execution
