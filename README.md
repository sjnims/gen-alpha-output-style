No cap, this marketplace contains a plugin that transforms Claude's responses to use Gen Alpha/brainrot internet slang. It's giving main character energy fr fr.

## Installation

In Claude Code, run:

```text
/plugin add sjnims/gen-alpha-output-style
```

Then follow the interactive prompts to select and enable the `gen-alpha-output-style` plugin. Start a new session and the style activates automatically.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [gen-alpha-output-style](./plugins/gen-alpha-output-style/README.md) | Transforms all responses to Gen Alpha slang |

## Example Output

**Before (normal Claude):**
> I'll help you fix this bug. The issue is on line 42 where the variable is undefined.

**After (with plugin):**
> Yo I got you fam, let me cook real quick. The issue is lowkey on line 42 where the variable said 'aight imma head out' and went undefined. No cap this is an easy fix fr fr.

## Features

- **Full brainrot mode**: Every response gets the Gen Alpha treatment
- **Technical accuracy preserved**: Code blocks stay clean and syntactically correct
- **Comprehensive glossary**: 50+ terms with meanings and usage patterns
- **Automatic activation**: Style applies via SessionStart hook
